## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Frankiiize/Basic-crud.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
